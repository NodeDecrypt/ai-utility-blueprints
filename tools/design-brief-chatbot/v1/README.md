# Design Brief Chatbot

Reusable tool definition for turning requirement summaries into stronger design briefs in Node Decrypt Community.

## Summary

Design Brief Chatbot is used as a reusable tool for turning requirement summaries into stronger design briefs.

This tool definition is public, reusable, and community-facing.

---

## Usage type

- `both`

This tool should be described in a community-safe way and reused across multiple blueprints.

---

## What this tool is good for

- turning requirement summaries into stronger design briefs
- turning rough inputs into clearer working material
- providing reusable support in multi-step blueprint flows
