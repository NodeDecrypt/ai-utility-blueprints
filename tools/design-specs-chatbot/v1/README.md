# Design Specs Chatbot

Reusable tool definition for turning design briefs into production-ready design specs in Node Decrypt Community.

---

## Summary

Design Specs Chatbot is used as a reusable tool for turning design briefs into production-ready design specs.

This tool definition is public, reusable, and community-facing.

---

## Usage type

- `both`

This tool should be described in a community-safe way and reused across multiple blueprints.

---

## What this tool is good for

- turning design briefs into production-ready design specs
- turning rough inputs into clearer working material
- providing reusable support in multi-step blueprint flows

---

## Resource folders

This pack includes `resources/single/` for reusable one-off guidance and `resources/manifests/` for grouped public-safe reference materials.
